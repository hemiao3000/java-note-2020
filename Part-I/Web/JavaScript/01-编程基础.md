# 基础知识

JavaScript 是一种可以用来给网页增加交互性的编程语言。

JavaScript 脚本也是一种程序，它们包含在

> - <small>HTML 页面内部（原先编写脚本的方式），或者</small>
> - <small>驻留在外部文件中（现在的首选方法）。</small>

```javaScript
<script type="text/javascript" src="js/hello.js"></script>
```

```javaScript
<script type="text/javascript">
    xxx;
    xxx;
    xxx;
</script>
```

在 HTML 页面上，因为脚本文本包围在 `<script>` 标签中，所以它不会显示在用户的屏幕上，而 Web 浏览器知道应该运行 JavaScript 程序。

`<script>` 标签常常放在 HTML 页面的 `<head>` 部分中。<small>但是如果愿意，也可以将脚本放在 <body> 部分中。</small>
