## String 类

|属性|说明|
|:-|:-|
|length|返回字符串中的字符数|


|方法|说明|
|:-|:-|
|`chatAt()`|以索引为参数，返回指定位置的字符|
|`indexOf()`|在字符串中查找指定字符串，返回它首次出现的索引位置|
|`lastIndexOf()`|在字符串中查找指定字符串，返回它最后一次出现的索引位置|
|`substring()`|返回两个索引之间的字符串，两个参数前闭后开|
|`split()`|以参数指定符号分割字符串，返回字符串数组|
|`trim()`|去掉字符串前后空格|
|`replace()`|在字符串中查找指定字符，替换成另一个参数。只替换第一个|
